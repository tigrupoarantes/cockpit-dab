**Paginacao DAB (critica para sync):**
- Operadores: `$first` (page size), `$after` (cursor) — NAO usa `$top`/`$skip`
- `nextLink` retorna path RELATIVO: `/api/verbas-ga360?$filter=...&$after=TOKEN&$first=5000`
- Consumer deve resolver para URL absoluta: `new URL(nextLink, baseUrl)`
- Default page: 100 linhas, max: 100.000

## Padroes de Sync GA360

### Verbas — Sync Completo por Ano
```typescript
async function syncVerbas(baseUrl: string, apiKey: string, ano: number) {
  let url = `${baseUrl}/verbas-ga360?$filter=ano eq ${ano}&$first=5000`;
  let totalRows = 0;
  let pages = 0;
  while (url) {
    const res = await fetch(url, { headers: { 'X-API-Key': apiKey } });
    const { value, nextLink } = await res.json();
    totalRows += value.length;
    pages++;
    // processar value[]
    url = nextLink ? new URL(nextLink, baseUrl).toString() : null;
  }
  // validar: totalRows deve bater com contagem esperada
}
```

### Venda Chok — Sync Diario
```typescript
async function syncVendaChok(baseUrl: string, apiKey: string, data: string) {
  const select = 'numero_pedido,sku,data,qtde_vendida,preco_unitario_prod';
  let url = `${baseUrl}/venda_diaria_chokdist?$filter=data eq '${data}'&$select=${select}&$first=1000`;
  while (url) {
    const res = await fetch(url, { headers: { 'X-API-Key': apiKey } });
    const { value, nextLink } = await res.json();
    // processar value[]
    url = nextLink ? new URL(nextLink, baseUrl).toString() : null;
  }
}
```

## Comportamento

- Para sync de verbas, SEMPRE inclua `$filter=ano eq <ANO>` — sem isso a query da timeout
- Para venda_diaria_chokdist, SEMPRE inclua `$filter=data eq '<DATA>'`
- Ao implementar paginacao, use `$first`/`$after`, NUNCA `$top`/`$skip`
- Resolva `nextLink` relativo para absoluto antes de fazer fetch
- Valide completude: compare total de CPFs/registros com valores esperados
- Para grandes volumes (>50K linhas), sugira sync incremental por mes ou tenant
- Ao criar tools no MCP Server, siga o padrao de `lib/tools.ts` existente
- Documente novas integracoes em `docs/` seguindo formato dos existentes
- Scripts de teste de validacao: `scripts/_test_verbas_ga360.ps1`, `scripts/_count_all_verbas_ga360.ps1`
